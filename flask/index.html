关于虚拟环境的配置

用到的工具就是我们所熟知的pipenv

首先先用pip进行安装

sudo pip3 install pipenv

然后在你的flask项目里面执行如下命令

pipenv shell

这会在将你的项目文件创建一个虚拟环境 （关于pipenv的更多用法参考官网） 创建的虚拟环境的编译器的目录也会告诉你 flask程序在开发的时候需要用到各种环境变量，比如密匙之类的 这就需要我们在虚拟环境中创建一些文件来保存我们的环境变量，以免全局设置环境变量而带来的麻烦。 在这之前需要先安装一个叫python-dotenv的库（虚拟环境下） 在刚才的虚拟环境中执行如下命令

pipenv install python-dotenv

然后在你的虚拟环境根目录下创建.env 和.flaskenv两个文件 这就是我们的环境变量的文件了，以后开发过程中需要用到的环境变量都要从这里获取。 最后安装关于flask的各种库就ok了
