# vscode-snippets

- 指定光标位置：$x
- 多光标编辑：$x $x
- 指定 placeholder 文本：${x:placeholder}
- 指定多选值：${x|aaa,bbb|}
- 取变量：$VariableName
- 对变量做转换：${VariableName/正则/替换的文本/i}
```text
"当前文件： $TM_FILENAME",
"当前日期： $CURRENT_YEAR/$CURRENT_MONTH/$CURRENT_DATE"
```
