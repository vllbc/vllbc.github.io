class Solution:
    def findContentChildren(g, s) -> int:
        g = sorted(g)
        s = sorted(s)
        n = 0
        for i in range(len(s)):
            if g[n] <= s[i]:
                n += 1
            if n == len(g):
                return n
        return n

皮尔逊相关系数

假设检验

步骤与概率论中假设检验的步骤一样。主要是统计量的构造

\[ t = r\sqrt{\frac{n-2}{1-r^2}} \] t为服从自由度为n-2的t分布

假设检验的条件

  1. 实验数据通常假设是成对的来自于正态分布的总体
  2. 实验数据之间差距不能太大
  3. 每组样本之间是独立抽样的。
